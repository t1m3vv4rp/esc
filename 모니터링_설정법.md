# 플레이어 답변 모니터링 설정 (구글 시트, 5분 컷)

플레이어가 입력하는 모든 답변(슬랙 답장, 문자, 신청서, 설문, 진행 단계)이
**실시간으로 내 구글 시트에 한 줄씩** 쌓이게 만드는 방법입니다.

---

## 1단계. 구글 시트 만들기
1. https://sheets.google.com → 새 스프레드시트 만들기
2. 이름은 아무거나 (예: "스토리랩 모니터링")
3. 1행에 머리글 입력: `수신시각` `플레이시각` `플레이어` `종류` `내용` `페이지`

## 2단계. Apps Script 붙여넣기
1. 시트 상단 메뉴 **확장 프로그램 → Apps Script** 클릭
2. 열린 편집기에서 기존 코드를 전부 지우고, 아래 코드를 붙여넣기:

```javascript
function doPost(e) {
  var sh = SpreadsheetApp.getActiveSpreadsheet().getSheets()[0];
  var d = {};
  try { d = JSON.parse(e.postData.contents); } catch (err) {}
  sh.appendRow([
    new Date(),
    d.t || "",
    d.player || "",
    d.type || "",
    JSON.stringify(d.data || {}),
    d.page || ""
  ]);
  return ContentService.createTextOutput("ok");
}
```

3. 상단 💾 저장

## 3단계. 웹앱으로 배포
1. 우측 상단 **배포 → 새 배포**
2. 톱니바퀴(유형 선택) → **웹 앱**
3. 설정:
   - 실행 계정: **나**
   - 액세스 권한: **모든 사용자** ← 중요! (익명 접속 허용해야 게임에서 전송됨)
4. **배포** 클릭 → 구글 계정 권한 승인 (경고 떠도 "고급 → 이동" 으로 승인)
5. 뜨는 **웹 앱 URL** 복사 (https://script.google.com/macros/s/……/exec 형태)

## 4단계. 게임에 URL 넣기
1. `telemetry.js` 파일을 열어 맨 위의:
   ```
   const NV_LOG_URL = "";
   ```
   따옴표 안에 방금 복사한 URL을 붙여넣기:
   ```
   const NV_LOG_URL = "https://script.google.com/macros/s/……/exec";
   ```
2. 저장 후 GitHub에 telemetry.js 덮어쓰기 업로드 → 끝!

## 확인
게임을 한 번 플레이해보면 시트에 실시간으로 줄이 쌓입니다.
휴대폰 구글 시트 앱으로도 실시간 확인 가능해요.

---

## 기록되는 것들
| 종류 | 내용 |
|---|---|
| game_start | 게임 시작 (통과한 이름) |
| name_reject | 화이트리스트에 없는 이름 입력 시 그 입력값 |
| chat | 슬랙에서 플레이어가 보낸 모든 답장 원문 |
| trigger | 발동된 키워드 (장난/농담/실제로/실시간/신기/무서) |
| chat_forced | "네 그럴게요"로 깨지기 전 원래 쓴 문장 |
| intro_done / mac_unlock / desk_done | 구간 통과 |
| imessage | 메시지 앱에서 노스텔지아에 직접 보낸 문자 |
| nv_apply | 신청서 전체 (이름·나이·꿈의 내용·신청 이유·설문 응답 포함) |
| nv_lookup | 조회 시도한 예약번호 |
| pencil_done / figure_done(실패횟수) / photos_done | 미니게임 |
| twist_start / scene3d_start / ending_reached | 후반 진행 |

## 참고
- URL을 안 넣으면(빈 문자열) 아무것도 전송되지 않고 게임은 그대로 작동합니다.
- 전송 실패해도 게임 진행에는 영향 없음 (조용히 무시).
- 시트 URL만 공유하지 않으면 나만 볼 수 있어요.
